Choir for WordPress helps you to listen to the activities, like page view, new comments and new posts on your WordPress site. 

== Description ==

Sound gives you ambient awareness without effort. You don't need to constantly switch contexts to see what's going on. 

== Installation ==

1. Install this plugin and activate it through the 'Plugins' menu in WordPress
1. Log into your [Choir](https://choir.io) account, create a WordPress data source and copy the API key
1. In your WordPress Admin interface, go to Plugins > Choir and paste in your API key.
